## エレベータピッチ(北極星)
「雨の日を楽しめない」課題を解決したい、
雨の中、外出している人向けの
「Wal:l:lamp」というプロダクトは
エンターテインメントかつアートなオブジェクトです。
これは雨が止まって見える空間を提供し、
スマホで音楽を聴いたり動画を見ながら歩くことと違って
受動的に雨の日だけの特別感が得られる。
