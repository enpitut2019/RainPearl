### 雨が止まる空間

## エレベータピッチ
「雨の日を楽しめない」課題を解決したい、
“雨だから“外出が憂鬱でめんどくさいと感じる人向けの
気持ちが晴れる「rain pearl(仮)」。
これは雨が止まって見える空間を提供し、
普段の雨の日の外出と違って
安全かつ受動的、雨の日だけの特別感が得られる。
